Multimodal Observer (MO)

MO project is an open source initiative intended to provide a low-cost solution for researchers and other professionals working with multimodal data. The MO project starts from a modular and scalable system that can aid the research process in the data collection, visualization, and analyses stages.

MO's architecture is based on plugins developed for different devices and data types. These plugins are integrated in a common environment, which enable users to work with multiple sources easily.

Through the MO project we invite developers, researchers, and professionals to contribute with ideas and plugins in order to produce an ecosystem of research tools for different purporses.
